## Sistema de Gestión de Sala de Cine (Java)
## Descripción
Este proyecto consiste en un sistema de gestión de una sala de cine desarrollado en Java, cuyo funcionamiento se realiza completamente a través de la terminal.
Permite administrar funciones de cine, asientos y venta de entradas mediante un menú interactivo.
## Contexto
Este proyecto fue desarrollado como parte de un trabajo académico del grado, con el objetivo de poner en práctica los 
conocimientos adquiridos durante el curso en la asignatura de programación.
La actividad consistía en diseñar e implementar una aplicación funcional en Java, 
utilizando programación con métodos.
Para ello, se eligió simular el funcionamiento de una sala de cine, permitiendo gestionar asientos
y la venta de entradas mediante un sistema interactivo por consola.
